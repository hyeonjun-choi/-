## 내용 3 - 리스트 작성

작성자가 원하는 만큼 목록을 줄지어 작성할수 있음을 확인. 앞의 기호(-,+,=)가 프리뷰에서 대분류, 소분류에 따른 기호(●,○)로 통일됨을 확인.
숫자를 넣은 채 목록 작성시 자동으로 숫자가 순차적으로 부여되어 정리됨을 확인

## 내용 4 - 돌아가기 경로 링크

## 내용 5 - 강세

특정 기호 *,_ 등을 통해 각각 기울이기, 굵게 강조 등으로 표현하여 강조할 수 있음을 확인.

*연습연습연습연습*
_연습연습여어어어언스으으읍_

**연습연습연습연습**
__연습연습여어어어언스으으읍__

_당신은 **가능** 조합한다 그것들을_

## 내용 6 - 블록 따옴표

> 
> 나 : 나를 낳으시고
> 
> 바 : 바지적삼
